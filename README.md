# NavaNirman

Hardware e-commerce website for Nawa Nirman Hardware.

## Included files

- `index.html` — Home page with hero, categories, featured products, brands, newsletter, and footer.
- `about.html` — About page with company mission, offerings, and values.
- `categories.html` — Categories page showcasing product groupings.
- `product.html` — Product detail page for the Cordless Drill Kit.
- `contact.html` — Contact page with company details and contact form.
- `styles.css` — Site stylesheet for layout, typography, responsive navigation, and page sections.
- `script.js` — Responsive navigation toggle logic for mobile menu buttons.

## Usage

Open `index.html` in a browser to view the homepage. The other pages are linked from the navigation menu for a complete site experience.
